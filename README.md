Кодовая кора для проектов на Unreal Engine включает в себя:
1. Машина инициализации состояний
2. Сервисная архитектура
3. Сохранения
4. Конфиги
5. MVVM Binders
6. Система экранов и попапов
7. Инъекция зависимостей
8. Система истории (квесты, туториал, сюжетные события)
9. Пул объектов
10. Система Handler (Actor, Pawn, Character)
11. Система для работы с ресурсами
12. Система твинов
